What is Leo?

Leo is an intelligent LLM-powered agent designed to increase event attendance and deepen user engagement on the Cuculi platform. Through sophisticated event-user matching, personalized outreach generation, and human-in-the-loop quality control, Leo operates as a semi-autonomous engagement engine.

How Leo Works

  1. Identify Target Events - Scan upcoming schedule and prioritize events
  2. Identify Target Users - Select users by theme and re-engagement potential
  3. Pair Events With Users - Match via topical alignment and preferences
  4. Generate Personalized Messages - Draft tailored outreach considering context
  5. Auto Quality Check - Run automated checks for quality and accuracy
  6. Human Review (HITL) - Present drafts to human reviewers for validation
  7. Export for Distribution - Format approved messages for scheduler
